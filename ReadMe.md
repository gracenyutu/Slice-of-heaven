## Description
{This is a web based application that allows the clients to;
* Choose the pizza of their choice
* Select the crust of their preference
* Choose as many toppings as desired
* Make an order online
* Choose if they want it delivered and where
* Get the total cost inclusive of delivery fee if they chose delivery
* Get the contact of the company incase of anything }
